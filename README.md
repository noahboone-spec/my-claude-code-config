# Claude Code Command Center

A visual showcase of my Claude Code power-user configuration — 11 MCP integrations, 25 custom commands, 12 plugins, 10 skills, 4 safety hooks, and 9 project workspaces.

## Live Page

View the landing page: [https://noahboone-spec.github.io/my-claude-code-config/landing-page.html](https://noahboone-spec.github.io/my-claude-code-config/landing-page.html)

## What's Inside

| Category | Count | Description |
|----------|-------|-------------|
| MCP Servers | 11 | Figma, Slack, Notion, Asana, Webflow, Apollo, Playwright, Exa, Knowledge Graph, Domo, Context7 |
| Plugins | 12 | GitHub, Frontend Design, TypeScript LSP, Pyright LSP, Code Review, and more |
| Skills | 10 | Domo API, Domo Branding, n8n workflows, Render Video |
| Commands | 25 | Design, media generation, research, Figma, and utility commands |
| Safety Hooks | 4 | Env commit block, production KG guard, SSR detector, credential detector |
| Custom Agents | 2 | Design Assistant, Google Agent Builder |
| Workspaces | 9 | Project-scoped configurations |
